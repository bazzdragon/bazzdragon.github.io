Ossi Heikkinen, game and web developer:

picture of me

I am a 17 year old game and web developer from Finland. I'm studying in OSAO vocational school on my second year the school will last 3 years. I have been interested in game development for a long time and I love to learn new tools and skills related to game development. I've always been interested in how games and websites are made and that's what got me into game development. I've been able to do many kinds of projects already and learn about all kinds of tools and different game genres. I have also been able to try different kinds of perspectives in games from 2D all the way to VR.

Here are all my projects

Adventures of Lobo

project image

Adventures of Lobo is a 2D platformer game where you play as a werewolf named Lobo. The game was my first ever game project and I made it alone using the Godot engine and GDscript. The game only features one level but it was a great introduction to game development. I tried to challenge myself by doing all kinds of stuff like falling and moving platforms. Game also has two kinds of enemies; those who go back and forth and one that activates only when you are close enough.

DEFEND.EXE

project image

This was my first project with PyGame. This was a few week summer project done in pairs. We made a tower defense game with my friend. In the game viruses are attacking your computer and you put down different virus defenses to stop them. When you try to place a tower you are presented with a question about coding or computers which you have to answer correctly to place the tower.
